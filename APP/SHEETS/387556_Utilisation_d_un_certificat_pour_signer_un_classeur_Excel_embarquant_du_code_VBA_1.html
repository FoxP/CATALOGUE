387556 — Utilisation d'un certificat pour signer un classeur Excel embarquant du code VBA — v1

Released Excel Autre Process excel classeur certificat cer installation carte puce macro désactivé sécurité signer

Problème

La base de données CATALOGUE étant basée sur du code VBA Excel, certaines fonctionnalités essentielles nécessitent que les macros soient activées par défaut. Initialement, un certificat a ainsi été créé pour que les avertissements de sécurité d'Excel ne bloquent plus le fonctionnement de l'outil. La présente fiche détaille l'installation ainsi que la désinstallation d'un certificat. Le problème suivant est susceptible d'être rencontré systématiquement à l'ouverture d'un classeur Excel non signé par un certificat, ou si ce même certificat n'est pas installé sur la machine : ﻿![Importation_certificat_0.png](FILES\387556_Utilisation_d_un_certificat_pour_signer_un_classeur_Excel_embarquant_du_code_VBA_1\Importation_certificat_0.png) Pour un classeur signé avec un certificat, sur une machine sur laquelle ce même certificat n'est pas installé, le message suivant peut apparaître : ﻿![Carte_à_puce.png](FILES\387556_Utilisation_d_un_certificat_pour_signer_un_classeur_Excel_embarquant_du_code_VBA_1\Carte_à_puce.png) La mise en place d'un certificat pour un classeur Excel embarquant du code VBA se fait en 4 étapes : 1. [Génération d'un nouveau certificat](#generate) 2. [Signature du classeur Excel avec le certificat](#sign) 3. [Export du certificat au format *.cer](#uninstall) 4. [Installation du certificat sur chaque machine](#install) A noter que la version actuelle de la base de données CATALOGUE ne nécessite pas de certificat, l'activation des macros est assurée par une option dédiée : ﻿![launcher_vbnet_1.png](FILES\387556_Utilisation_d_un_certificat_pour_signer_un_classeur_Excel_embarquant_du_code_VBA_1\launcher_vbnet_1.png) ﻿![launcher_vbnet_2.png](FILES\387556_Utilisation_d_un_certificat_pour_signer_un_classeur_Excel_embarquant_du_code_VBA_1\launcher_vbnet_2.png) ﻿![launcher_vbnet_3.png](FILES\387556_Utilisation_d_un_certificat_pour_signer_un_classeur_Excel_embarquant_du_code_VBA_1\launcher_vbnet_3.png)

1. Installation d'un certificat :

___ Double cliquer sur le fichier *.cer du certificat (exemple : ﻿[CATALOGUE.cer](FILES\387556_Utilisation_d_un_certificat_pour_signer_un_classeur_Excel_embarquant_du_code_VBA_1\CATALOGUE.cer)) pour l'installer : ﻿![Importation_certificat_1.png](FILES\387556_Utilisation_d_un_certificat_pour_signer_un_classeur_Excel_embarquant_du_code_VBA_1\Importation_certificat_1.png) ﻿![Importation_certificat_2.png](FILES\387556_Utilisation_d_un_certificat_pour_signer_un_classeur_Excel_embarquant_du_code_VBA_1\Importation_certificat_2.png) ﻿![Importation_certificat_3.png](FILES\387556_Utilisation_d_un_certificat_pour_signer_un_classeur_Excel_embarquant_du_code_VBA_1\Importation_certificat_3.png) ﻿![Importation_certificat_4.png](FILES\387556_Utilisation_d_un_certificat_pour_signer_un_classeur_Excel_embarquant_du_code_VBA_1\Importation_certificat_4.png) ﻿![Importation_certificat_5.png](FILES\387556_Utilisation_d_un_certificat_pour_signer_un_classeur_Excel_embarquant_du_code_VBA_1\Importation_certificat_5.png) ﻿![Importation_certificat_6.png](FILES\387556_Utilisation_d_un_certificat_pour_signer_un_classeur_Excel_embarquant_du_code_VBA_1\Importation_certificat_6.png) ﻿![Importation_certificat_7.png](FILES\387556_Utilisation_d_un_certificat_pour_signer_un_classeur_Excel_embarquant_du_code_VBA_1\Importation_certificat_7.png) Selon la politique de sécurité mise en place, il peut être nécessaire de répéter l'opération pour les magasins de certificats ci-dessous. A l'installation d'un certificat, l'option "Sélectionner automatiquement le certificat en fonction du type de certificat" ne fonctionne que très rarement. ﻿![Importation_certificat_8.png](FILES\387556_Utilisation_d_un_certificat_pour_signer_un_classeur_Excel_embarquant_du_code_VBA_1\Importation_certificat_8.png) ___

3. Générer un certificat :

___ Exécuter l'utilitaire `Digital Certificate for VBA Projets` : ![création_certificat_1.png](FILES\387556_Utilisation_d_un_certificat_pour_signer_un_classeur_Excel_embarquant_du_code_VBA_1\création_certificat_1.png) S'il n'est pas accessible depuis le menu démarrer, l'exécutable correspondant est le suivant : `SELFCERT.EXE`. ﻿![création_certificat_4.png](FILES\387556_Utilisation_d_un_certificat_pour_signer_un_classeur_Excel_embarquant_du_code_VBA_1\création_certificat_4.png) Renseigner le nom du certificat à créer puis valider : ﻿![création_certificat_2.png](FILES\387556_Utilisation_d_un_certificat_pour_signer_un_classeur_Excel_embarquant_du_code_VBA_1\création_certificat_2.png) Le certificat est désormais créé : ﻿![création_certificat_3.png](FILES\387556_Utilisation_d_un_certificat_pour_signer_un_classeur_Excel_embarquant_du_code_VBA_1\création_certificat_3.png) Un classeur Excel signé avec un certificat n'affichera aucun avertissement de sécurité sur chaque machine sur laquelle le même certificat a été installé. ___
